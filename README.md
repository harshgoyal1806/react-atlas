**React Atlas** is a modern web application built using React and Vite that lets you explore country information from around the world. You can search for any country, view detailed information, and sort countries based on different parameters such as population, region, or name. The app provides a fast, responsive, and intuitive interface to explore countries easily.

## Features

- Search for countries by name.
- Sort countries by population, name, region, and other parameters.
- View detailed information about each country.
- Responsive and clean user interface.
- Fast and efficient with Vite as the build tool and React for UI.

## Technologies Used

- React
- Vite
- Tailwind CSS
- REST Countries API
